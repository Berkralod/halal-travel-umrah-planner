## FAQ

### What endpoints does the Halal Travel & Umrah Planner API include?
The API provides 10 key endpoints: prayer times lookup, halal hotel search, Umrah pilgrimage guide, Quran reference database, dua library, halal food checker with barcode scanning, Islamic finance information, Muslim city scores, and additional travel planning features designed specifically for Islamic requirements.

### How accurate are the prayer times provided by this API?
Prayer times are calculated using established Islamic methods and updated in real-time based on astronomical data and geographical coordinates. The API supports multiple calculation conventions (Hanafi, Shafi'i, etc.) to accommodate different Islamic schools of thought.

### Can I use the barcode scanning feature to check if food products are halal?
Yes, the halal food checker endpoint allows barcode scanning to instantly verify products' halal status. The system uses AI/ML technology trained on extensive halal certification databases, ingredient analysis, and Islamic dietary guidelines to provide accurate assessments.
